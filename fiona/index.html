輪胎基本認識

TIRE BASIC UNDERSTANDING

  • 輪胎品牌 Brand logo
  • 最大載重及胎壓 Max Load & Pressure
  • 安裝方向 (Inside, Ouside) / Mounting direction
  • 強化輪胎標示 Extra Load
  • 輪胎的規格 Tire specifications
  • 製造產地 Made in
  • 輪胎花紋名稱 Eagle F1 Asymmetric 3
  • 3T Treadwear 耐磨指數 / Traction 溼地牽引力
    / Temperature 溫度指數
  • 製造日期(周/年) Manufacture date
  • 規格代碼 Manufacturer code
  • 美國運輸部縮寫 Dot
  • 無內胎之輪胎 Tubless
  • 輻射層輪胎 Radial
